# storiesディレクトリのルール

## ルール

`src/stories/` ディレクトリには以下のようなファイルを配置する。

### 配置するもの
- **UIライブラリ関連のストーリー**: Ant Designなどの外部ライブラリのUIコンポーネントのStorybookファイル
- **ラップしていない状態のコンポーネント**: プロダクト固有のロジックやスタイリングを適用していない、純粋なUIコンポーネントのストーリー
- **プロダクト内で使用しているものに限る**: 実際にアプリケーションで使用されているコンポーネントのみ

### 配置しないもの
- プロダクト固有のロジックが含まれたコンポーネントのストーリー（これらは該当コンポーネントの近くに配置）
- ビジネスロジックが含まれたコンポーネントのストーリー

## ディレクトリ構造
```
src/stories/
├── antd/          # Ant Designコンポーネントのストーリー
├── theme/         # テーマ関連のストーリー
└── ...            # その他のUIライブラリのストーリー
```
